Website Project Site Plan

Site Name

Delta Rental Hub

This name was selected because it clearly represents a central platform where rental properties in Delta State can be listed and discovered by users.

Site Purpose

The purpose of this website is to provide a platform where landlords in Delta State can showcase rental properties and where users can browse available houses. The site will display property listings with images, location, price, and descriptions. It will also provide a way for users to make inquiries about available properties.

Scenarios

  • Where can I find affordable houses for rent in Delta State?
  • How can I contact a landlord about a property I am interested in?

Color Scheme

Primary Color (#0077cc): Used for headings, buttons, and highlights.

Secondary Color (#f4f7f9): Used for background and section areas.

Typography

Poppins: Used for headings.

Roboto: Used for body text.

Wireframe

Hero Section

Desktop View
